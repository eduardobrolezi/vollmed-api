VollMed API — API RESTful para gestão de clínica médica (projeto desenvolvido no curso da Alura)


Desenvolvi a VollMed API durante o curso de Spring Boot 3 da Alura, aplicando na prática os principais conceitos de desenvolvimento backend com Java. 
A aplicação oferece endpoints para cadastro e autenticação de usuários, além de operações CRUD completas voltadas à rotina de uma clínica médica (como médicos, pacientes e agendamentos), 
seguindo os princípios de uma API REST bem estruturada.

Tecnologias e padrões utilizados:

Java 17 + Spring Boot 3

Spring Web para construção da API REST

JPA/Hibernate para persistência em banco relacional

Arquitetura organizada em camadas (Controller, Service, Repository)

Maven e Maven Wrapper para gestão de dependências

Git + GitHub para versionamento e boas práticas de organização de projeto

Esse projeto demonstra minha capacidade de estruturar um backend do zero, aplicar boas práticas de arquitetura, persistência e modelagem, além de consolidar minha experiência com o ecossistema Spring. 
Continuo evoluindo o repositório como parte do meu portfólio e do meu processo contínuo de aprendizado.
